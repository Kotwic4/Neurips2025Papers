# Neurips2025Papers

[Website](https://kotwic4.github.io/Neurips2025Papers/)

Unofficial personal browser for [NeurIPS 2025 papers](https://neurips.cc/). Cluster labels and textual annotations (e.g. summaries, “explain like I’m 5”) are derived from [Jay Alammar’s NeurIPS 2025 visualization](https://jalammar.github.io/assets/neurips_2025.html) .
